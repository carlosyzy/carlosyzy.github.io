ShaderAsset 着色器资产

概述

ShaderAsset（着色器资产）用于存储着色器代码，包括顶点着色器和片段着色器。

头文件

#include "core/asset/shader-asset.h"

API 接口

构造函数

接口说明
ShaderAsset()创建空着色器资产
ShaderAsset(uuid)通过 UUID 创建着色器资产

着色器创建

接口说明参数返回值
create(type, glslData, macros)通过 GLSL 代码创建着色器type: 着色器类型，glslData: GLSL 代码，macros: 宏定义void

着色器类型

常量名说明
ShaderVertexAsset顶点着色器
ShaderFragmentAsset片段着色器
