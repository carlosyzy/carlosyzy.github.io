MaterialAsset 材质资产

概述

MaterialAsset（材质资产）用于存储材质的配置数据，包括顶点着色器、片段着色器、纹理、渲染状态等信息。

头文件

#include "core/asset/material-asset.h"

API 接口

构造函数

接口说明
MaterialAsset()创建空材质资产
MaterialAsset(builtin, uuid, path)通过内置材质创建

材质创建

接口说明参数返回值
create(materialData)通过 JSON 数据创建材质materialData: JSON 格式的材质数据void

着色器设置

接口说明参数返回值
setVertexShader(vert)设置顶点着色器vert: ShaderAsset 指针或路径void
setFragmentShader(frag)设置片段着色器frag: ShaderAsset 指针或路径void

纹理设置

接口说明参数返回值
setTexture(texture)设置主纹理texture: TextureAsset 指针void
setTexture(key, texture)设置指定键的纹理key: 纹理键，texture: TextureAsset 指针void
