Component 组件系统设计文档

Component（组件）是 Boo 引擎中用于扩展节点功能的可插拔模块。组件必须挂载到节点（Node）上才能工作。

组件类型

  • Layer2D 组件：用于 Node2D 节点（如 UISprite、UIText）
  • Layer3D 组件：用于 Node3D 节点（如 MeshRenderer）

组件层级

值说明
Default默认层级
Layer2D2D 组件层
Layer3D3D 组件层

API 接口

基础信息

接口说明返回值
getUuid()获取组件 UUIDconst string
getName()获取组件名称string
getNode()获取所属节点Node*
getLayer()获取组件层级EComponentLayer

激活控制

接口说明参数返回值
isEnabled()获取自身启用状态-bool
setEnabled(enabled)设置启用状态enabled: true/falsevoid
isEnabledInHierarchy()获取层级启用状态-const bool

生命周期

接口说明
OnAwake()组件唤醒函数（纯虚函数）
OnEnable()组件启用函数（纯虚函数）
Update(deltaTime)组件更新函数（纯虚函数）
LateUpdate(deltaTime)组件延迟更新函数（纯虚函数）
OnDisable()组件禁用函数（纯虚函数）
OnDestroy()组件销毁回调函数（纯虚函数）

自定义组件实现

引擎内组件

#pragma once
#include <string>
#include "core/component/component.h"
#include "core/component/component-register.h"

namespace Boo
{
    class MyComponent : public Component
    {
    public:
        MyComponent(std::string name, Node *node, std::string uuid = "");

        void OnAwake() override;
        void OnEnable() override;
        void Update(float deltaTime) override;
        void LateUpdate(float deltaTime) override;
        void OnDisable() override;
        void OnDestroy() override;
        void destroy() override;
        void setProperty(json &data) override;

        ~MyComponent() override;
    };

    REGISTER_COMPONENT(MyComponent, "My Custom Component")
}

引擎外组件（用户项目）

#pragma once
#include <string>
#include "core/component/component.h"
#include "core/component/component-register.h"

class MyComponent : public Boo::Component
{
public:
    MyComponent(std::string name, Boo::Node *node, std::string uuid = "");

    void OnAwake() override;
    void OnEnable() override;
    void Update(float deltaTime) override;
    void LateUpdate(float deltaTime) override;
    void OnDisable() override;
    void OnDestroy() override;
    void destroy() override;
    void setProperty(json &data) override;

    ~MyComponent() override;
};

REGISTER_COMPONENT(MyComponent, "My Custom Component")
